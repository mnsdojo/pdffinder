# pdffinder

Search, download, and scan PDFs from the web.

## Install

```bash
pipx install pdffinder
```

Or with pip:

```bash
pip install pdffinder
```

## Usage

```bash
pdffinder search "quantum computing"
pdffinder download "https://example.com/paper.pdf"
pdffinder scan "paper.pdf"
```

## Development

```bash
pip install -e .
```
